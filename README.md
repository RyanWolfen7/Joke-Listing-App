## Installation

- Create a AUTH0 
- create a dot env 
```
touch .env
```
- Fill in your Auth0 info
```
export AUTH0_API_CLIENT_SECRET=""
export AUTH0_CLIENT_ID=""
export AUTH0_DOMAIN=""
export AUTH0_API_AUDIENCE=""

```
